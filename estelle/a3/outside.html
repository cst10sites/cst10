Inner Tags
Tag Definition Example
<link> Links the definition of the current page to an external one. Useful for external CSS. Uses a rel and href tag. This page uses the link tag for external CSS!
<style> Allows for internal or external CSS, which can elevate the look of your website. This page was decorated using mostly external CSS.
font-family Defines the font used for the selected text. Font names with spaces require quotations around them.
This text is in a different font.
color Sets the font colour for the selected text. This can be chosen using hex codes, colour names, or rgb(a).
This font has a different colour.
background-image Sets the background to a linked image. The background of this page uses this tag.
